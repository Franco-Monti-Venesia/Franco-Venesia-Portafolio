Mis proyectos

E-commerce Gerardino Pastas

Tienda online en funcionamientro desarrollada para una fábrica de pastas artesanales, con el objetivo de digitalizar su proceso de ventas y mejorar la experiencia del cliente.

Este proyecto permite navegar productos por categorías, ver detalles, armar un carrito de compras, seleccionar cantidad, completar un checkout personalizado. Además, se incluye un resumen visual del pedido y confirmación por email automático.

  • JavaScript
  • CSS
  • HTML
  • Firebase
  • Bootstrap

App de control de gastos personales

Aplicación web desarrollada para llevar un registro ordenado y visual de ingresos y egresos mensuales. El objetivo es ayudar a los usuarios a tomar decisiones financieras más conscientes.

Permite cargar operaciones con nombre, monto, tipo y categoría, filtrarlas dinámicamente, editarlas o eliminarlas, y visualizar el balance general actualizado en tiempo real.

  • CSS
  • HTML
  • Tailwind CSS
